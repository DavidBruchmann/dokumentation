# Kategorien

Kategorien sind die Container für alle Inhalte Ihrer Website. Typischerweise entspricht eine Kategorie auch einem Menüpunkt der Website im Frontend. Setzen Sie Kategorien daher sinnvoll und geordnet ein, um ihre Inhalte zu sortieren. Kategorien haben daher meist Namen wie „Unternehmen“, „Produkte“ oder „Kontakt“, wo Sie dann weitere Unterkategorien oder gleich Artikel mit Inhalten zu Ihrem Unternehmen erstellen können.
